| Concepto                                           | En SQL                                                                     | En Python                                                                                                     |
| :------------------------------------------------- | :------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------ |
| **Seleccionar todos los elementos**                | `SELECT * FROM tabla;`                                                     | `[x for x in tabla]`                                                                                          |
| **Seleccionar una columna**                        | `SELECT nombre FROM estudiantes;`                                          | `[nombre for nombre, nota in estudiantes]`                                                                    |
| **Filtrar filas (WHERE)**                          | `SELECT * FROM estudiantes WHERE nota >= 4.0;`                             | `[e for e in estudiantes if e["nota"] >= 4.0]`                                                                |
| **Seleccionar + filtrar (proyección + condición)** | `SELECT nombre FROM estudiantes WHERE nota >= 4.0;`                        | `[nombre for nombre, nota in estudiantes if nota >= 4.0]`                                                     |
| **Transformar valores (SELECT con expresión)**     | `SELECT UPPER(nombre) FROM estudiantes;`                                   | `[nombre.upper() for nombre, nota in estudiantes]`                                                            |
| **Asignar alias (AS)**                             | `SELECT nombre AS estudiante FROM alumnos;`                                | `[{"estudiante": nombre} for nombre in alumnos]`                                                              |
| **Filtrar con múltiples condiciones (AND / OR)**   | `SELECT * FROM ventas WHERE total > 1000 AND cliente = 'Aldo';`            | `[v for v in ventas if v["total"] > 1000 and v["cliente"] == "Aldo"]`                                         |
| **JOIN / combinaciones**                           | `SELECT * FROM clientes JOIN pedidos ON clientes.id = pedidos.cliente_id;` | `[(c, p) for c in clientes for p in pedidos if c["id"] == p["cliente_id"]]`                                   |
| **Agrupar y resumir (GROUP BY + COUNT)**           | `SELECT ciudad, COUNT(*) FROM clientes GROUP BY ciudad;`                   | `{ciudad: len([c for c in clientes if c["ciudad"] == ciudad]) for ciudad in {c["ciudad"] for c in clientes}}` |
| **Ordenar resultados (ORDER BY)**                  | `SELECT * FROM productos ORDER BY precio DESC;`                            | `sorted(productos, key=lambda p: p["precio"], reverse=True)`                                                  |
| **Diccionario clave-valor (SELECT col1, col2)**    | `SELECT id, nombre FROM usuarios;`                                         | `{u["id"]: u["nombre"] for u in usuarios}`                                                                    |
